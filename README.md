# Player-API Getter

Este meu projeto é uma simples API em java utilizando String Boot para retornar estatísticas de um usuário em seu Servidor.

## Pré-requisitos

- JDK 17
- XAMPP (MYSQL)
- IntelliJ (Ou outro IDEA)

## Tutorial

Para rodar o site no IntelliJ basta ir na classe 'Application' e apertar o botão de run, depois disso basta colocar no seu navegador localhost:8080/player/nick (Lembrando que você terá que alterar de acordo com o que tem em seu banco de dados, altere as informações do seu banco como database, senha etc em 'src/main/resources/application.properties' e os dados que você vai querer puxar em Users e claro adapte para o UserResponse UserRequest e DataController)
